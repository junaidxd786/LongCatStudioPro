## üöÄ Features

-   **Multi-Format Script Generation**: Support for YouTube Videos, TikTok/Reels, Stories, Podcasts, Educational Tutorials, and News Reports.
-   **Advanced AI Models**: Integrated with LongCat AI models (Flash Chat, Flash Thinking, Flash Lite) for diverse content needs.
-   **Tone & Style Control**: Customize content tone (Energetic, Casual, Edgy, Narrative, Humorous, Authority).
-   **Viral Hook Generator**: Automatically generate 5 click-worthy hooks for any topic.
-   **Script Analysis**: Real-time word count, estimated read time, and structural analysis (Hook/CTA detection).
-   **Quick Refinements**: One-click tools to Shorten, Expand, Simplify, or "Make Viral" your scripts.
-   **Modern UI/UX**: Sleek, dark-mode interface built with Tailwind CSS and Lucide Icons.

## üì¶ Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/junaidxd786/LongCatStudioPro.git
    cd LongCatStudioPro
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Configure Environment Variables**
    Create a `.env` file in the root directory and add your LongCat API key:
    ```env
    VITE_LONGCAT_API_KEY=your_api_key_here
    ```

4.  **Start the Development Server**
    ```bash
    npm run dev
    ```

## üìñ Usage

1.  Open the application in your browser (usually `http://localhost:5173`).
2.  Enter a **Topic** in the left sidebar.
3.  Select your desired **Format**, **Tone**, and **Length**.
4.  (Optional) Toggle options like **Viral Hooks**, **Timestamps**, or **Visual Cues**.
5.  Click **Generate** to create your script.
6.  Use the **Refine** tools at the bottom to tweak the output.
